# 🍽️ Restaurant Website

A responsive multi-page Restaurant Website built using **Node.js**, **Express.js**, and **EJS**. The project demonstrates server-side rendering, routing, and modern web design principles through a simple restaurant-themed website.

## 🚀 Features

- 🏠 Home Page with welcome section
- ℹ️ About Page describing the restaurant's mission
- 📞 Contact Page with restaurant details
- 🧭 Navigation Bar for seamless page navigation
- 🎨 Responsive and user-friendly UI
- ⚡ Server-side rendering using EJS
- 🔗 Express.js routing implementation

## 🛠️ Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- HTML5
- CSS3

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/debanjana253048-hub/Restaurant-Website.git
```

### 2. Navigate to Project Directory

```bash
cd Restaurant-Website
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run the Application

```bash
node index.js
```

### 5. Open in Browser

```text
http://localhost:3000
```
